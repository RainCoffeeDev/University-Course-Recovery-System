# Course Recovery System (CRS)

A Java-based application designed to manage course recovery processes with distinct user roles and concurrent management features.

## 🚀 Getting Started

Follow these steps to get the project up and running on your local machine:

1. **Download the Project:** Download the `.zip` file of this repository and extract it to your preferred directory.
2. **Open in IDE:** Open **Apache NetBeans**.
3. **Load the Project:** Go to `File` > `Open Project`, navigate to the extracted folder, and select the Java Project.
4. **Run:** Clean and build the project, then hit the **Run** button.
